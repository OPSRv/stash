# Stash

**Менюбар-хаб для дрібних задач — усе, що не варто окремого застосунку, живе в одному попапі.**

## Опис

Stash — це macOS menubar-застосунок, який замінює десяток маленьких утиліт однією іконкою в треї. Попап 920×520 викликається глобальним шорткатом `⌘⇧V`, між модулями перемикаєшся через `⌘⌥1..9`. Усе офлайн-first, без акаунтів і хмари — дані лежать локально.

Філософія — свідомий «Франкенштейн»: краще одна компактна скринька з інструментами під рукою, ніж зоопарк застосунків у доку. Кожен модуль самодостатній, вмикається ліниво, не заважає іншим.

## Модулі

- **Clipboard** — історія буфера обміну з пошуком, пінами і швидким повторним копіюванням
- **Downloader** — завантаження відео й аудіо з YouTube та інших джерел, авто-детект посилання з буфера, прев'ю, черга
- **Notes** — нотатки з rich-редактором, вбудованими аудіо-записами й зображеннями, AI-чатом у кожній нотатці
- **Translator** — швидкий переклад між мовами прямо з попапу
- **AI** — чат з LLM для щоденних запитів, пояснень, генерації тексту
- **Music** — локальний аудіо-плеєр для фонової музики без перемикання на Spotify
- **Metronome** — метроном для музичних практик із налаштуванням темпу й розміру
- **Pomodoro** — таймер за технікою Pomodoro з нотифікаціями та історією сесій
- **Terminal** — вбудований термінал для швидких команд без переходу в iTerm
- **Settings** — глобальні шорткати, accent-колір, мова інтерфейсу, папки за замовчуванням
